═══════════════════════════════════════════════════════════════════════════
   HOW TO USE
═══════════════════════════════════════════════════════════════════════════

STEP 1: Navigate to Monthly Statements
────────────────────────────────────────
  Click on "Monthly Statements" in the navigation bar (third tab)

STEP 2: Find the Month You Want to Explore
────────────────────────────────────────────
  Scroll down to the "Monthly cashflow trend" table
  Look for the month you're interested in (e.g., "Nov 2024")

STEP 3: Click "View Transactions"
──────────────────────────────────
  Click the "View Transactions" button for that month

STEP 4: View Filtered Transactions
───────────────────────────────────
  The app automatically:
  ✓ Switches to the "Transactions" view
  ✓ Shows ONLY transactions from that month
  ✓ Pre-fills the date filters with the month's start and end dates

   TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Q: I clicked "View Transactions" but nothing happened
A: Refresh the page and try again. Ensure JavaScript is enabled.

Q: The transactions shown don't match the monthly totals
A: Some transactions may have been edited or deleted. Check the original
   data source (CSV files) for discrepancies.

Q: Can I filter by multiple months at once?
A: Not directly. Use the date range filter in the Transactions view to
   manually select a multi-month range (e.g., Jan 1 - Mar 31).

Q: Where did the old "View Details" button go?
A: It's still there! Both buttons are now shown in the Actions column.
