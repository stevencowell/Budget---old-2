## Color Coding

### Priority Levels
```
🔴 High Priority    = Red    = Immediate action needed
🟡 Medium Priority  = Yellow = Review this week
🔵 Low Priority     = Blue   = Review when convenient
```

### Status Badges
```
🟢 Active    = Green = Regularly charging
🟠 Warning   = Orange = Overdue for charge
⚪ Inactive  = Gray   = Not charged recently
```

### Spending Indicators
```
Positive amounts (income)    = Green text
Negative amounts (expenses)  = Orange/Red text
Neutral/info                 = White/Gray text
```

## Tips for Best Visual Experience

1. **Use a modern browser** (Chrome, Firefox, Safari, Edge)
2. **Enable JavaScript** (required)
3. **Allow localStorage** (for saving rules)
4. **Use desktop for full experience** (mobile works but less optimal)
5. **Zoom:** 100% recommended
6. **Resolution:** 1366x768 minimum, 1920x1080 optimal
